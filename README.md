###### Installation
1. Clone the repository 
<pre> git clone https://github.com/jqjacq/Drinks_Ecommerce.git </pre>
2. Install NPM packages using `npm install`
<pre> npm install </pre>
3. Start the project using `npm start`
<pre> npm start </pre>
